## (Tic-Tac-Toe)

A simple and interactive X-O (Tic-Tac-Toe) web game where two players can challenge each other locally on the same device.

## Features

- Two-player mode
- Winning & draw detection
- Scoreboard (optional)
- Clean and responsive UI
